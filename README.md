# Mimi Markov Chain

Generates a few sentences using a markov chain with [mimihatesyou.com](http://mimihatesyou.com/) as the source dictionary.

## Setup

    $ bundle install

## Usage

Generate twenty (the default number) sentences:

    $ ruby mm.rb

Generate ten sentences:

    $ ruby mm.rb -n 10
